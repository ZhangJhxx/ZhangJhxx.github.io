## Quick Start
### 初始化站点 `hexo init`

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Run server

``` bash
$ hexo server
$ hexo s
```
### cache clean
``` bash
$ hexo clean
$ hexo c
```
### Generate static files

``` bash
$ hexo generate
$ hexo g
```

### Deploy to remote sites

``` bash
$ hexo deploy
$ hexo d
```
